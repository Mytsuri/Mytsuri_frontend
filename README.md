# Mytsuri_frontend 리드미

## 프로젝트 개요
- 축제 정보 탐색, 리스트 관리, 알림, 지도 기반 탐색 등 다양한 기능을 제공하는 프론트엔드 SPA입니다.

## 1. 기획 및 설계
- React + Vite 기반 SPA
- 컴포넌트 기반 구조, 페이지별 폴더 분리
- 상태 관리: useState, useEffect 등 React 내장 훅 활용
- API 연동: axios(fetch)로 백엔드와 통신
- 반응형 UI, Tailwind CSS 또는 커스텀 CSS

## 2. 주요 페이지 및 기능
- 홈, 축제 목록/상세, 리스트, 지도, 알림, 프로필, 온보딩 등
- 리스트에 축제 추가/삭제, 초대, 편집 등
- 알림 확인 및 읽음 처리
- 지도 기반 축제 탐색

## 3. API 연동 예시
- GET `/api/festivals` : 축제 목록 불러오기
- POST `/api/users/login` : 로그인
- GET `/api/notifications` : 알림 목록

## 4. 기술 및 구현의 어려움
- 다양한 페이지/컴포넌트 상태 동기화
- API 에러 및 로딩 처리
- 모바일/데스크탑 반응형 UI 구현
- 지도 연동 및 마커 최적화
- 리스트/알림 등 실시간성 요구

## 5. 해결 방법
- 공통 컴포넌트 및 커스텀 훅 분리
- 에러/로딩 상태 전역 관리
- CSS 미디어쿼리, flex/grid 활용
- 지도 API(카카오/네이버 등) 커스텀 마커 적용
- 알림/리스트 변경 시 useEffect로 실시간 반영

## 6. 실행 방법
1. `npm install`로 의존성 설치
2. `npm run dev`로 개발 서버 실행
